# Client-Side Form Validation using React

A simple **Login Form with Client-Side Validation** built using **React.js**.  
This project validates user input such as **Email** and **Password** before allowing form submission.

---

## 🚀 Features

- Email validation using Regular Expressions
- Password strength validation
- Real-time error messages
- Controlled form inputs using React state
- Clean and responsive UI
- Prevents invalid form submission
- Form reset after successful submission

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- React Hooks (`useState`)
